## Description
[cite_start]Ce projet consiste à automatiser l'analyse des données de ventes pour une entreprise de e-commerce utilisant Python et VS Code

## Travail Réalisé
* [cite_start]**Génération de données** : Création d'un fichier `ventes.csv` avec les colonnes ID, Prix, Quantite, et Remise.
* **Calculs automatiques** : 
    * [cite_start]Calcul du **Chiffre d'Affaires Brut** (Prix * Quantité).
    * [cite_start]Application des **remises** pour obtenir le **CA Net**.
    * [cite_start]Calcul de la **TVA (20%)** sur le CA Net.
* **Analyse** : 
    * [cite_start]Affichage du **CA Total** de l'entreprise.
    * [cite_start]Identification du produit le plus rentable.
* [cite_start]**Exportation** : Création du fichier `resultats_final.csv` contenant tous les calculs.
* [cite_start]**Bonus** : Visualisation des données avec **Matplotlib**.

## Instructions
1. Exécuter `generer_ventes.py` pour initialiser les données.
2. Exécuter `analyse_ventes.py` pour effectuer les calculs et voir les graphiques.
